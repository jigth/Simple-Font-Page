Simple Font Page

This is a simple font page, made with Roboto Slab and Roboto Medium. It's simple Enough to have just typographies and demonstrate the CSS @Fontface property at the same time

If you want to see this page clearly, please disable DarkMode temporary as it has no support for that right now

As for the HTML, it have been formated with Prettier, automatic formating is usually better and way easier than doing all of that manually. It's great to be able to focus on what I need to achieve and not too much on the looks of my code, just make it clean enough to be understandable by people.

Now lets just add some nonsense to enjoy this typography.
